# 问题-整改-效果 三联页模板

## 模板字段
- 问题描述：
- 影响范围：
- 触发接口/模块：
- 修复方案：
- 验证方式：
- 结果数据/截图：

---

## 示例 1：鉴权白名单过宽
- 问题描述：`/**/query` 被放行导致潜在未授权读取。
- 影响范围：多查询接口。
- 修复方案：精确白名单替代通配符。
- 验证方式：未登录访问受保护查询返回认证失败。
- 结果：未授权读取路径关闭。

## 示例 2：商品更新越权
- 问题描述：非商品 owner 可提交他人商品 id 更新。
- 修复方案：`owner/admin` 校验。
- 验证方式：普通账号更新他人商品被拒绝。
- 结果：横向越权修复。

## 示例 3：文件路径穿越
- 问题描述：`fileName` 参数可被构造为目录穿越。
- 修复方案：文件名安全校验 + canonical 路径边界检查。
- 验证方式：异常路径请求返回 400。
- 结果：文件读取边界收敛。
