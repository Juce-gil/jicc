# 5分钟系统演示脚本（答辩版）

## 0:00 - 0:40 项目背景
- 本系统面向校园二手交易场景，采用 Vue3 双前端 + Spring Boot 后端。
- 重点解决“预约面交”闭环和权限安全问题。

## 0:40 - 1:50 主流程演示（学生端）
1. 学生登录（账号认证成功）。
2. 浏览商品并发起预约。
3. 在“我的订单”查看待确认状态。

## 1:50 - 2:50 主流程演示（卖家侧）
1. 切换卖家视角查看收到的预约。
2. 确认预约后状态变为 RESERVED，商品锁定。
3. 双方确认完成后订单变 COMPLETED，商品 SOLD。

## 2:50 - 3:40 管理端演示
1. 管理员登录。
2. 查看仪表盘统计、用户/订单/日志查询入口。
3. 展示管理查询均受角色限制。

## 3:40 - 4:40 安全整改亮点
- 放行规则由通配改为精确白名单。
- 商品、内容、互动删除新增 owner/admin 归属校验。
- 文件下载新增路径穿越防护。

## 4:40 - 5:00 总结
- 系统已具备可运行、可演示、可测试的毕业设计条件。
- 下一步将补齐校园认证/举报/公告等扩展模块。
